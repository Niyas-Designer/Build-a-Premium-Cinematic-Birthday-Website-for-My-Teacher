# 🌙 Birthday Story Website

A modern, interactive birthday website built with **React**, **Vite**, and **Tailwind CSS**. This project transforms a traditional birthday greeting into a cinematic storytelling experience with smooth animations, interactive elements, and a clean, responsive design.

## ✨ Features

* 🎬 Interactive storytelling experience
* 🐱 Animated cartoon character
* 🎂 Creative birthday celebration
* 🎧 Optional voice narration
* ✨ Smooth animations & transitions
* 📱 Fully responsive design
* 🌙 Modern dark UI
* ♿ Accessibility-friendly

## 🛠 Tech Stack

* React
* Vite
* TypeScript
* Tailwind CSS
* Framer Motion
* GSAP
* Lenis Smooth Scroll

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/birthday-story.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
